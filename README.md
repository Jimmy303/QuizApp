Installeer Node.js en npm (https://nodejs.org/en/download)

voer de volgende commando uit om Angular CLI the installeren
npm install -g @angular/cli

Voer de volgende commando uit om de dependencies van het project te installeren (uitvoeren in het project map)
npm install

Voer de volgende commando uit om de app de starten
ng serve

Voer de volgende commando uit om unittest uit te voeren
ng test
